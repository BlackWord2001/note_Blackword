# Linux

### 基础指令
|指令|作用|
|:-|:-
|`mkdir`|创建文件夹
|`tree`|查看当前文件目录的树形图
|`cp`|复制文件
|`realpath`|查看文件路径
|`pwd`|显示当前目录完整路径
|`rm`|删除；`-r`删除目录；`-f`强制删除

### 解压缩文件
+ 压缩指定文件夹 <br>
`tar -zcvf filename.tar.gz dir filename`

### Vim
|指令|作用
|:-|:--
|:q|退出
|:q!|强制退出
|:w|保存
|:qw|保存并退出
|:q!|不保存文件强制退出
|:wq!|强制保存并退出
|:e!|放弃所有修改，从上次保存文件开始再编辑
